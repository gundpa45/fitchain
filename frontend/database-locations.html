Configuration Files:

📁 Project Root ├── backend/ │ ├── config/ │ │ └── database.js ← Database configuration │ └── view-database.js ← Database viewer script └── database-viewer.html ← Web-based database viewer

Database Schema:

📁 Project Root └── backend/ └── database/ └── schema.sql ← Database schema definitions

🗂️ Database Tables Structure

Table Name Purpose Key Fields
users User accounts and profiles id, wallet_address, username, created_at
contests Fitness contests and challenges id, title, start_date, end_date, prize_pool
activities User fitness activities id, user_id, distance, duration, gps_data
leaderboard User rankings and scores id, user_id, total_distance, rank
payments Stellar payment transactions id, from_wallet, to_wallet, amount, tx_hash
prize_distributions Contest prize payouts id, contest_id, user_id, amount, status
